# wodfeeder
I was too lazy to open my Gym webpage everyday. Too many clicks. So I thought it would be a good idea to scrap it webpage and display all the week's exercises in a single page, with just one click.

The project is running at https://wodfeeder.herokuapp.com

Also, since the exercises are being stored by date on the database, iti is possible to access them individually by the url

 - https://wodfeeder.herokuapp.com/wod/<date iso formate>.
  
For example: https://wodfeeder.herokuapp.com/wod/2019-05-08
  
Also, the project has a news feeder, in the case you want to sign it up. Its address is https://wodfeeder.herokuapp.com/feed.
