# wodfeeder
I was too lazy to open my Gym's webpage everyday. Too many clicks. So I thought it would be a good idea to scrap it and display all the weekly exercises in a single page, with just one click.

The project is running at https://wodfeeder.herokuapp.com

Also, since the exercises are being stored by date on the database, it is possible to access them individually by the url

 - https://wodfeeder.herokuapp.com/wod/<date iso formate>.
  
For example: https://wodfeeder.herokuapp.com/wod/2019-05-08
  
If I not even even want to access the webpage, the project has a news feeder which I can sign up. The address is https://wodfeeder.herokuapp.com/feed.
